# 💻 Site GalaxyTours 

Este é um projeto realizado para uma avaliação formativa de front-end, focada na inserção de Media Query no arquivo de CSS, e demonstra um site para uma agência de turismo espacial.

## 🛠️ Tecnologias Utilizadas
* HTML5
* CSS

## 📁 Estrutura do Projeto
Organização do Projeto::
/
├── index.html
└── src/
    ├── styles/  
        └── style.css
    ├── scripts/ 
        └── script.js
    └── images/  
        └── foguete.png

## 🚀 Funcionalidades Implementadas
* **Media Queries (Resolução adaptada):** Foram implementadas no CSS do código dois pontos de quebra para tornar o site mais responsivo, sendo eles o de tablet com uma largura máxima de 768px (Mínima de 481px), e por fim o ponto de quebra para Smartphones com uma largura máxima de 480px. Com estes pontos de quebra o layout não quebrará mais nestes dispositivos.
