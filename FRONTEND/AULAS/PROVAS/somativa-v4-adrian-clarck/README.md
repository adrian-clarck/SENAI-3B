# 💻 Site CinePipoca 

Este é um projeto realizado para uma avaliação de front-end, focada na inserção de scripts, DOM e Eventos DOM, e demonstra um site para um Cinema.

## 🛠️ Tecnologias Utilizadas
* HTML5
* CSS
* JavaScript

## 🚀 Funcionalidades Implementadas
* **Interatividade Visual:** Foi implementado um evento de entrada e saída de mouse para o Banner Vip, quando o mouse entra no banner ativa o modo vip e quando sai o modo vip sai junto.
* **Saudação Dinâmica:** Uma mensagem que altera de acordo com a hora do usuário, dizendo que libera uma das 3 sessões especificas (Manhã, Tarde, Noite).
* **Cálculo em Tempo Real:** Na compra de bilhetes foi implementado um evento de input que calcula em tempo real o preço da compra de ingressos, dependendo do valor de ingressos que serão comprados, que é informado pelo usuário.
* **Injeção de Conteúdo:** Foram adicionados dois eventos de Clique, aonde o primeiro, com o botão de Publicar Filme, insere na tela a informação escrita pelo usuário, e o segundo, o botão de Limpar Mural, que apaga a informação inserida anteriormente e o retorna para a caixa de escrita.
